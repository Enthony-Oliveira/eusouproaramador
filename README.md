Eu Sou Programador

Este é um projeto desenvolvido em React Native (JavaScript), inspirado no clássico app “Eu Sou Rico”.
A ideia do aplicativo é introduzir, de forma prática, os conceitos fundamentais do React Native, como:

View

Text

Image

StyleSheet

É um projeto perfeito para quem está começando no desenvolvimento mobile e deseja entender a estrutura básica de uma interface no React Native.

Sobre o App

O aplicativo exibe:

Um header personalizado com o título Eu Sou Programador

Uma imagem centralizada na tela

Um layout simples e organizado, ideal para quem está iniciando

Esse projeto ajuda a aprender na prática como:

Organizar componentes com View

Exibir textos com Text

Renderizar imagens com Image

Estilizar elementos com StyleSheet
